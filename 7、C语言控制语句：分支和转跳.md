##### 赋值运算符：=

```c
bmw=2002
```

赋值运算符=号左侧是一个变量名，右侧是赋给该变量的值。

C语言中以下语句无意义

```c
2002=bmw
```

这种情况下，2002称为右值(rvalue)，只能是字面常量。不能给常量赋值，常量本身就是他的值，因此编写代码=号左侧必须是一个变量名，实际上赋值运算符左侧必须引用一个存储位置，最简单的方法就是一个变量名。简单来说，C使用可修改的左值(modifiable lvalue)标记哪些可赋值的实体。
